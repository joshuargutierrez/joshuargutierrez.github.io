  • User Stories

  • User Story:

    As a player of Custom Hero RPG Survival I want to upgrade my character, so I can try different builds.

    Acceptence Criteria:

    Ensure the characters in the game are able to individually have:
    A player level
    Abilities
    A talent
    A weapon, or unarmed fighting style

    User Story:

    As a player of Custom Hero RPG Survival I want to be able to play with my friends online.

    Acceptence Criteria:

    Ensure the the game has:
    A client-server model
    Network connectivity
    The ability for players to interact with each other

    User Story:

    As a player of Custom Hero RPG Survival I want to be able to choose different characters.

    Acceptence Criteria:

    Ensure that:
    The user can choose different heros
    Each hero is unique
    Hero's should individually be capable of different animations.

    User Story:

    As a player of Custom Hero RPG Survival I want to be able to use different weapons.

    Acceptence Criteria:

    Ensure that:
    Individual players can choose different weapons for their heros
    Weapons can be customized
    Weapons contain animations
    Weapons can be equipped

    User Story:

    As a player of Custom Hero RPG Survival I want to sell items

    Acceptence Criteria:

    Ensure the the game has:
    A store for purchases/sales
    Create a monetary system for the game
    Introduce a profitability factor in combat

    User Story:

    As a player of Custom Hero RPG Survival I want to be a woman, man, robot, etc.

    Acceptence Criteria:

    Ensure the the game has:
    Multiple genders and species for the characters
    multiple ethnicities for the characters
    Individualized personalitiesm for each character
